# 🧨 rooijakkers.software.full-stack

Frontend and backend for vending machine.

## 📋 Prerequisites

- java
- npm

## 🏗 Build

```sh
make build
```

## 🏃 Run

```sh
make run
```

## 🔥 Visit

Navigate to [localhost:8080][1] for the frontend.
